How To Install Plugin

Installing Subscribe To Download is very simple and only takes a few seconds.
 
1. In your WordPress dashboard go to your Pugins page and add a new plugin using the upload option.
 
2. Select the wp-freebie-subscribe.zip file NOT the file you downloaded from CodeCanyon. If you upload the wrong zip file you will get an error and the plugin will not install.
 
3. Click Activate and you are all set! Continue on to the settings to configure the plugin.

General Settings

Bit.ly Setup - Here you can setup link shortening with Bit.ly for the download / verification links. Go to your Bit.ly account page and get your API key and add it to the API Key setting field in the plugin settings, save, and you are ready to go! You can also add a custom shortener domain if you have one setup on Bit.ly

Fake Email Penalty - Enter the number of minuets users should wait if they entered a fake email they could not verify. Default is 3.

Template Settings

Extra Fields - Here you can choose to enable or disable the extra form fields. If you choose to enable a field, the field will be required to submit the form.

Headings And Messages - You will also find a slew of headings and messages you can customize to fit your needs. These changes will take effect anywhere you use the plugin and it's shortcode. 

Email Setup - Here you can setup the email that gets sent to verify the user. The from name and email address control who the email will appear to be sent from, and you can also edit the subject line of the email. You can also edit the message that gets sent in the email, the verification / download link will be added at the end of the message.

 

How To Use

From your post / page editor you will see a new email icon in your tinyMCE toolbar. This MCE plugin will generate the shortcode for you. All you need to do is insert the download URL and the rest will be taken care of for you.

Keep in mind that download URLs should push the file to the user, or you could use an uploading service. If the URL does not start a download, the user will instead be redirected to the URL you provide.

It's that simple! The plugin will handle everything else and build your list in the background.

Export / Manage List

Under the Subscribe to DL menu you will find the List Manager page. This page allows you to view an prune your list, as well as filter your list.

To export the list data to a service like MailChimp or Awebber, find out which format you need. 99% of services will use or allow CSV files. The plugin will allow you to export your entire list data with one click from the List Manager. Simply click the Export As CSV or Export as TXT buttons at the top of the List Manager page to export.

The export data will contain all data possible for each user including extra field data.

Subscriber Stats

If you want to see how your list is going, the Analytics tab in the settings panel will give you a great breakdown. You can see how many subscribers you are gaining by day, week, month, year, and compair them to previous time frames.
